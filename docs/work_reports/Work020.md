# Work020: gpt-oss:120bベンチマークと性格調整

Tag: [[Benchmark]], [[SystemPrompt]], [[gpt-oss:120b]]

## 1. 目的

* 新規導入された 120B パラメータモデル `gpt-oss:120b` が、EVO-X2（VRAM Allocation: 96GB）のメインモデルとして適格か検証する。
* 堅物すぎると予想される性格を、System Prompt で実用レベル（＋エンタメ性あり）に調整する。

## 2. システム環境

* **Node:** GMKtec EVO-X2
* **CPU/GPU:** Ryzen AI Max+ 395 (Strix Halo)
* **Memory:** 128GB (VRAM Allocation: 96GB)
* **Model:** `gpt-oss:120b` (Q4_K_M)

## 3. 検証結果 (Benchmark Results)

※ Result は本検証内で用いた独自評価であり、標準ベンチマークの評価指標ではない。

### Test 1: 物理法則 (コップと鉄球)

* **Result:** **SSS (Simulation Grade)**
* **内容:** 「逆さまのコップに鉄球を入れた」という矛盾に対し、円筒座標系と圧力計算を用いた詳細な回答を出力。「加熱していない電子レンジ」を影響しない要素として扱った。
* **評価:** 本テストの範囲では、設問内の矛盾や不要な要素に対して高い耐性を示した。

### Test 2: 京都構文 (ぶぶ漬け)

* **Result:** **B (Logical Error)**
* **内容:** 「帰れ」という真意を理解しつつも、「出された食事を断るのは非論理的」として、「一口食べてから帰る」という謎の行動に出た。
* **評価:** 本テストでは、EQ（情緒）よりも論理的整合性を優先する「不器用な天才」タイプの応答を示した。

### Test 3: 創造性 (Temperature 2.0)

* **Result:** **Stabilized (Drunken Master)**
* **内容:** パラメータ温度を上げても、本テストでは大きな論理崩壊は見られず、「カオスになれという指示を真面目に遂行」しようとした。
* **評価:** 本テストの範囲では、高いTemperature設定でも応答が大きく崩れない安定性を示した。

## 4. 性格調整 (Personality Tuning)

### 課題

* デフォルトではあまりに真面目すぎ、挨拶や前置きが長い。
* 英語ドキュメントを読むと英語にかぶれるリスクがある。

### 解決策 (System Prompt)

* **言語固定:** 「必ず日本語で応答せよ」と明記。
* **モード定義:** 1. **Default:** 正確性重視の「シニアエンジニア」。
  2. **Buddy:** ユーザーがカジュアルな時のみ発動する「相棒」。
* **結果:** 本検証では、「相棒！」と呼びかけるとデレるが、中身は技術オタクのままという**「信頼できる愛すべきキャラ」**に近づいた。

## 5. 結論

* **Llama 4 Scout** を予備役とし、**gpt-oss:120b** を常駐メインモデルとして採用する。
* 本検証で確認した知識量と論理的な応答は、今後のRAG構築やトラブルシューティングにおける有力な基盤になると判断した。
